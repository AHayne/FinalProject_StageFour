# Personal Budget Tracker

**Short description:**  
A simple personal budget tracker that records income & expenses, categorizes transactions, and shows totals. Built in Python; runs from `src/main.py`.

Features
- Add/edit/delete transactions (amount, category, date, type)
- View summary totals (income, expenses, balance)
- Save/load transactions to/from JSON file

Requirements
- Python 3.8+ (no external dependencies required)
- (Optional) `pip install -r requirements.txt` if you add lib

Run the app
From the project root:
```bash
python src/main.py
