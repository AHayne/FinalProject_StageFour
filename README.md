Personal Budget Tracker

Overview
The Personal Budget Tracker is a Python application that allows users to record, view, edit, and delete financial transactions. This will helps user manage their spending by organizing transactions into categories, calculate totals, and generating summaries.  
The project is designed for clarity, convieniece, and easy expansion.

Features
- Add new income or expense transactions  
- Edit or delete existing transactions  
- Categorize transactions (Food, Rent, Entertainment, etc.)  
- View all transactions in a clean menu-driven interface  
- View total balance, income, and expenses  
- Data saved to a file so the user's information persists  
- Fully menu-based text application run through `main.py`
